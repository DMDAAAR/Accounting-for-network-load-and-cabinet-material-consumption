# Система учёта сетевой инфраструктуры и расхода материалов (Кабинет 319Б)

Проект (Диагностика и модернизация ЛВС) Предназначен для фиксации сетевых точек, учёта текущих дефектов оборудования с возможностью загрузки фотоотчётов и контроля фактического расхода материалов.

## 🚀 Стек технологий
- **Backend:** PHP 7.4+ / 8.x (Процедурный стиль, безопасное подключение через PDO)
- **Database:** MySQL 5.7+ / 8.0
- **Frontend:** HTML5, CSS3, Bootstrap 5 (через CDN)

---

## 🏗️ Архитектура проекта (Файловый MVC)

Код приложения строго разделен по принципу **Model-View-Controller (MVC)** без использования сторонних фреймворков, что обеспечивает чистоту и читаемость структуры:



### Структура каталогов:
```text
infrastructure-project/
│
├── controllers/          # Контроллеры страниц (.controller.php)
├── models/               # Модели для работы с БД (.model.php)
├── views/                # HTML-шаблоны страниц (.views.php)
├── uploads/              # Папка для загружаемых фотографий дефектов
└── connectDB.php         # Подключение к базе данных
└── index.php             # Точка входа (перенаправление на дашборд)
